What We Offer

Route Selection

Provided a source or destination explicitly or on map you could get a number of matching transport solutions

Retrieve Current Location

Whenever the user reaches our website he shall be notified of his current location so as to assist in deciding a source or the exact destination

Reaching For an Airport Cab

There is a database of all the standalone taxis and all the taxi companies for reaching airport as well as other places by providing complete info about the taxis

Search Nearby Places

Given a specific location you could retrieve a list of nearby places with some breif and basic info about that landmark

Notifications

The system has builtin functionality for user to recieve sms notifications about any route at any given time by maintaining his search logs as all times

Searching a short path

The user shall also be provided with options to view the shortest paths on a map giving any two locations as inputs , by using longitudes, latitudes and google maps in background
